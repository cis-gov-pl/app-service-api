Source Code
-----------

The source code is available through git repository::

    git clone ssh://<user_name>@usrint.cis.gov.pl/mnt/home/kklimaszewski/GitRepos/WebServices
    cd WebServices
    git submodule init
    git submodule update

Updating the code to the latest version::

    git pull
    git submodule update
